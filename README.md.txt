# St. Mary's Church Website

A minimalist, responsive website for St. Mary's Anglican Church in Mandalay, Myanmar. This is a complete static site that requires no backend or database.

## Features

- **Responsive Design**: Works perfectly on all devices from mobile to desktop
- **Accessibility**: WCAG AA compliant with proper semantic markup
- **Fast Performance**: No heavy frameworks, optimized images, minimal JavaScript
- **Easy Content Management**: Simple file-based article system
- **Dark/Light Theme**: Automatic theme toggle with user preference saving
- **SEO Optimized**: Proper meta tags, Open Graph, and semantic structure

## Getting Started

### Quick Setup

1. **Download/Clone**: Get all the files in this repository
2. **Upload**: Upload all files to your web hosting service
3. **Access**: Visit your domain to see the live site
